Веб-сайт інтернет-магазину аксесуарів, створений на базі фреймворку Django. Проєкт включає систему авторизації, особистий кабінет, корзину покупця та автоматичний імпорт товарів через зовнішнє API
Стек технологій: Python, Django, SQLite, Bootstrap 5, HTML/CSS.

Щоб розгорнути проєкт на своєму комп'ютері, виконайте наступні кроки у терміналі

Крок 1: Завантаження проєкту
Якщо ви завантажили проєкт ZIP-архівом, просто розпакуйте його і відкрийте папку в PyCharm або VS Code. Якщо використовуєте Git, виконайте:

Bash
git clone <посилання-на-ваш-репозиторій>
cd <назва-папки-проєкту>
Крок 2: Створення віртуального середовища
Щоб не засмічувати систему, створюємо ізольоване середовище для бібліотек проєкту:

Bash
python -m venv venv
Активуйте віртуальне середовище:

Windows: venv\Scripts\activate

macOS / Linux: source venv/bin/activate

Крок 3: Встановлення залежностей
Коли з'явиться префікс (venv), встановіть необхідні бібліотеки (Django та requests для роботи з API):

Bash
pip install django requests
(Якщо у вас є файл requirements.txt, ви можете використати команду pip install -r requirements.txt)

Крок 4: Створення бази даних (Міграції)
Оскільки база даних не зберігається на GitHub, нам потрібно створити таблиці на основі наших моделей (Користувачі, Товари, Корзина):

Bash
python manage.py makemigrations
python manage.py migrate
Крок 5: Наповнення бази товарами
У проєкті є спеціальний скрипт, який автоматично завантажує актуальні аксесуари з API та зберігає їх у вашу локальну базу даних. Запустіть його:

Bash
python manage.py import_products
Крок 6: Запуск сервера
Тепер все готово! Запускаємо локальний сервер:

Bash
python manage.py runserver
Відкрийте браузер і перейдіть за адресою: http://127.0.0.1:8000/

💡 Що робити далі?
Натисніть "Реєстрація", щоб створити свого першого користувача (для зручності тестування складність паролів вимкнено, можна використовувати прості паролі, як 1234).

Після входу ви зможете додавати товари в корзину та переглядати каталог.
